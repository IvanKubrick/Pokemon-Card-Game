About the game

It is a matter of common knowledge, that match-match-game is a favorite game for all generations. It is easy to play, in fact it is so simple that really young children can play with ease. It requires observation, concentration and a good memory to win.

The aim of match-match-game is to find pairs for all cards using as little time and cards as possible.

How to play

  1. Press play to start a new game.
  2. Choose proper card style and difficulty (number of cards) from the lists provided.
  3. Press start to run a game.
  4. Select two cards and if they match they will disappear. In other case they will turn back.
  5. Your task is to find a pair for each pokemon as soon as possible.
